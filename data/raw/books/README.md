# Textbook PDFs (local only)

Place textbook PDFs here for a **full** knowledge base build. These files are **not** distributed via Git (copyright).

Suggested references (obtain legally yourself):

- Jurafsky & Martin — *Speech and Language Processing* (3rd ed.)
- Manning & Schütze — *Foundations of Statistical NLP*
- Silge & Robinson — *Text Mining with R*
- Bengfort et al. — *Applied Text Analysis with Python*

Then run:

```bash
python -m knowledge_base.build_kb
```

For the public demo, use **papers only** (see `data/raw/papers/`).
